## 安装node.js

参考：http://t.csdn.cn/PMaC6

## 使用idea写vue

在idea中安装插件：vue.js

# 样例

```html
<!doctype html>
<html class="no-js" lang="" xmlns:v-bind="http://www.w3.org/1999/xhtml">

<head>
  <meta charset="utf-8">
  <title>初始Vue</title>
  <script type="text/javascript" src="js/vue.js"></script>
</head>

<body>
  <!--
    1. 插值语法
        功能：解析标签体内容
        写法：{{}}
    2. 指令语法
        功能：解析标签（包括标签属性，标签体内容，绑定事件...）
  -->
  <div id="root">
    <h1>Hello {{name}}</h1>
    <!--v-bind: 可以简写为 : -->
    <a v-bind:href="url">百度一下</a>
    <a :href="url">搜一下</a>
  </div>

  <script type="text/javascript">
    //阻止vue在启动时生成生产提示。
    Vue.config.productionTip = false
    //这个要放在对应的模块后面，如果放在div前面，那么{{}}的内容就不会被解析。
    new Vue({
      el:'#root',
      data: {
        name:"张三",
        url:'https://www.baidu.com'
      }
    })
  </script>
</body>
</html>
```
